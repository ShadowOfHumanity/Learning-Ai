# **Functions & Models**

### **A Model Is Just A Function**
Example: Scalar (1D) Linear Regression
```
f(x) = w*x + b
```
in vector form: (nD) Linear Regression
```
f(x) = w . x + b    (w dot x plus b)
```

**This is linear regression ^^**
